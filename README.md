# PHP1
## Изменится ли вывод?
# Да
bool(true)
int(12345)
bool(false)
bool(false)

bool(true)
int(12345)
bool(false)
bool(true)

Если test.php содержит var_dump(...), вывод может отличаться, особенно при сравнении типов (===, ==) или в работе некоторых функций, так как поведение PHP иногда меняется между версиями.
